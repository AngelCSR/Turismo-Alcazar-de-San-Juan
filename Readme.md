# Turismo Alcázar de San Juan

## Descripción del Proyecto

**Turismo Alcázar de San Juan** es una página web creada con fines educativos para promocionar los principales atractivos turísticos de la ciudad de Alcázar de San Juan, situada en la región de Castilla-La Mancha, España.  

El proyecto combina diseño moderno, contenido multimedia y una estructura organizada para ofrecer al visitante información clara y atractiva sobre lugares emblemáticos como los **Molinos de Viento**, la **Casa del Hidalgo** y el **Parque Alces**. Además, incluye una sección de **contacto** para que los usuarios puedan enviar consultas o sugerencias.  

---

## Características Principales

- **Diseño Responsive:** Adaptable a móviles, tabletas y escritorio gracias a Bootstrap 5.  
- **Secciones Destacadas:**  
  - Portada con encabezado visual.  
  - Presentación de los lugares más emblemáticos con imágenes y descripciones.  
  - Video y audioguía de los Molinos de Viento.  
  - Mapa interactivo de Alcázar de San Juan usando Google Maps.  
- **Formulario de contacto:** Con validación básica y alerta personalizada al enviar el mensaje.  
- **Estilo uniforme:** Todas las páginas siguen la misma estética, márgenes, tipografía y colores.  
- **Footer consistente:** Información de copyright y proyecto educativo.

---

## Tecnologías Utilizadas

- **HTML5** para la estructura de la página.  
- **CSS3** y un archivo `estilo.css` para el diseño personalizado.  
- **Bootstrap 5** para el diseño responsive y componentes predefinidos (navbar, botones, formularios).  
- **JavaScript** básico para alertas y redirección tras enviar el formulario.  
- **Google Maps Embed** para mostrar la localización de Alcázar de San Juan.  

---

## Cómo Ejecutar el Proyecto

1. Clonar o descargar el repositorio en tu máquina local.  
2. Abrir los archivos `.html` en cualquier navegador moderno (Chrome, Firefox, Edge, Safari).  
3. Navegar entre las páginas usando la barra de navegación para explorar los contenidos.  

> **Nota:** El formulario de contacto simula el envío mediante JavaScript y muestra una alerta. No está conectado a un servidor real para recepción de correos.
